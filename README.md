### Desafio inicial

- Criar um repositório
- Dar os comandos iniciais do git em uma pasta do computador
  - `git init`
  - `git add (. ou -A)`
  - `git commit -m "..."`
  - Na primeira vez tu tem que dar `git add remote caneca <<link do repositorio>>`
  - `git push caneca master`
